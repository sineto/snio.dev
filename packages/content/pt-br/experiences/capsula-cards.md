Capsula Cards é a primeira plataforma brasileira de flashcards dedicada à residência médica, idealizada por médicos aprovados nas melhores instituições do país.

##### A plataforma foi composta por quatro camadas, todas desenvolvidas e mantidas por mim nas primeiras versão em produção e com número relevante de usuários ativos diariamente:

- Infraestrutura — AWS (EC2, ECS, RDS, Lambda Functions, Amplify), Terraform, Docker, Shell Script. Ambientes de staging e produção provisionados com foco em menor custo e máxima disponibilidade.

- Backend — REST API desenvolvida com Node.js, TypeScript, Express e PostgreSQL. Integração com Stripe para pagamentos e Firebase para autenticação e armazenamento.

- Frontend — Aplicação web construída com Next.js, integrada à API e ao Firebase.

- Backoffice (CMS) — Solução de gestão de conteúdo desenvolvida em ReactJS, permitindo que a equipe editorial publicasse e gerenciasse flashcards de forma autônoma.

- CI/CD implementado com GitHub Actions, Docker e AWS CLI — automatizando ciclos de teste, build e deploy para staging e produção.

- Stack: TypeScript · Node.js · Express.js · ReactJS · Next.js · Docker · Firebase · Stripe · AWS · Terraform
