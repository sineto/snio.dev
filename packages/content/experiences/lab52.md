Atuei como desenvolvedor full-stack em múltiplos produtos digitais ao longo de quase 3 anos, evoluindo de Júnior para Pleno e assumindo progressivamente responsabilidades de arquitetura e infraestrutura.

##### Competências:
- **Capsula Cards** — Primeira plataforma brasileira de flashcards para residência médica: Como único desenvolvedor do projeto nas primeiras versões de produção, assumi decisões importantes de arquitetura e fui responsável end-to-end pela plataforma — da infraestrutura em produção ao frontend — ao longo de 1 ano e 4 meses. Provisionei os ambientes de staging e produção em AWS (EC2, ECS, RDS, Lambda, Amplify) com Terraform e Docker, implementei a REST API, a aplicação web, o backoffice CMS, pagamentos com Stripe, autenticação com Firebase e pipelines de CI/CD com GitHub Actions.

- **Jurídico.ai** — Plataforma de IA generativa para advogados (baseada na API da OpenAI):
Desenvolvi o frontend com ReactJS, Zustand e Material UI seguindo padrões mobile-first. Contribuí com novas funcionalidades na API backend em Go + go-chi + PostgreSQL.

- **LegalCloud** — Plataforma de gestão para escritórios de advocacia:
Desenvolvimento de funcionalidades e integração entre times de design, produto e engenharia.

- Ao longo dos três contextos, fui referência na ponte entre design e desenvolvimento — garantindo consistência entre protótipos e implementação final, e colaborando diretamente com líderes de produto e estratégia.

- Stack: Node.js · TypeScript · Go · ReactJS · Next.js · PostgreSQL · AWS · Terraform · Docker · Stripe · Firebase · GitHub Actions
