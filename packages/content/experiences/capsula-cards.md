- Estive envolvido em todas as partes do processo de desenvolvimento das aplicações que compõe a
plataforma: Infraestrutura de staging/produção em Cloud, REST API, CMS (Content Management
System) e aplicação Web.
- Principal responsável por provisionar a infraestrutura, de staging e produção, em Cloud, visando
sempre o menor custo possível em implementação e deploy das aplicações. As ferramentas mais
utilizadas nesse estágio do processo foram Shell Script, Docker, Terraform, e serviços do provedor
AWS como EC2, ECS, RDS, Amplify, Lambda Functions;
- Principal responsável por desenvolver também as primeiras versões da REST API e da solução de
back-end para servir tanto a aplicação web quanto a aplicação de backoffice. TypeScript, Node.js,
Express.js e PostgreSQL foram as tecnologias usadas para no processo de desenvolvimento;
- Principal responsável por desenvolver as primeiras versões da aplicação Web usando Next.js, e
também de toda a solução de backoffice (o CMS) da plataforma para gerenciamento de conteúdo
com ReactJS;
- Desenvolvi as primeiras soluções de pagamento, de autênticação e autorização para a base de
clientes da plataforma usando Stripe como processador de pagamento e Firebase para Auth e
storage;
- Fui um dos principais responsáveis por implementar e manter pipeline CI/CD para ambientes de
staging e produção, com ciclos de testes, build e merge automatizados após revisão de código, onde
foram usadas como ferramentas para auxiliar no processo GitHub Actions, Docker e AWS CLI.
