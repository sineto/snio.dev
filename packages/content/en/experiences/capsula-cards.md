Capsula Cards is Brazil's first flashcard platform dedicated to medical residency, created by doctors who were accepted into the country's top medical institutions.

##### The platform was composed of four layers, all developed and maintained by me during its first production versions with a relevant number of daily active users:

- Infrastructure: AWS (EC2, ECS, RDS, Lambda Functions, Amplify), Terraform, Docker, Shell Script. Staging and production environments provisioned with a focus on lowest cost and maximum availability.

- Backend: REST API built with Node.js, TypeScript, Express and PostgreSQL. Integrated with Stripe for payments and Firebase for authentication and storage.

- Frontend: Web application built with Next.js, integrated with the API and Firebase.

- Backoffice (CMS): Content management solution built in ReactJS, enabling the editorial team to publish and manage flashcards autonomously.

- CI/CD implemented with GitHub Actions, Docker and AWS CLI: automating test, build and deploy cycles for staging and production.

- Stack: TypeScript · Node.js · Express.js · ReactJS · Next.js · Docker · Firebase · Stripe · AWS · Terraform
